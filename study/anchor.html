a 태그

<a>

  • 다른 페이지를 연결할 때 사용하는 하이퍼링크(hypertext reference)를 정의할 때 사용
  • href 속성을 이용해 link의 목적지를 가리킨다. href 속성이 없다면, target, download, rel, type 속성들도 사용할 수 없다.
  • 방문하지 않은 링크(:link): 파란색
  • 방문했던 링크(:visited): 보라색
  • 활성화된 링크(:active): 빨간색
  • HTML 4에서는 하이퍼링크, 앵커 둘 다 사용 가능했지만, HTML5에서는 하이퍼링크 용도로만 사용되며, href가 없는 경우에는 하이퍼링크를 위한 표시 역할만을 한다.
<a href="http://www.tcpschool.com" target="_blank"></a>
