Element

<font>

  • HTML4에서는 font face, font size, color를 지원한다.
  • HTML에서 <font>은 실제 크기가 작다.
  • font 태그는 HTML5에서 지원하지 않는다. CSS로 처리하는 걸 추천
<font>HTML5에서 더 이상 지원하지 않으며, CSS로 처리하는 걸 추천합니다.</font>
필수 속성
속성명 속성값 설명
color 색상 텍스트 색상을 16진수로 설정하거나 명명된 색상으로 설정
face 문자 하나 이상의 글꼴 이름 목록을 쉼표로 구분해서 표현. 기본 스타일의 문서 텍스트는 브라우저가 지원하는 첫번째 글꼴로 렌더링 된다.
size 크기 글꼴 크기를 숫자 또는 상대 값으로 지정
브라우저 지원
지원 지원 지원 지원 지원 지원
