<h1> ~ <h6>

  • HTML 제목을 정의하는 데 사용된다.
  • h1이 가장 중요한 제목을 정의하고 h6이 가장 덜 중요한 제목을 정의한다.
  • h1은 페이지 당 하나만 사용한다.
  • 스크린 리더 사용자가 흔히 사용하는 탐색 기법은 제목에서 제목으로 뛰어넘으며 페이지 콘텐츠를 빠르게 파악하는 것이다.
    따라서 제목 단계를 뛰어넘으면 중간에 빠진 제목이 어디 있는 건지 모르므로 사용자의 혼란을 야기할 수 있습니다.
<h1>제목 h1</h1>
<h2>제목 h2</h2>
<h3>제목 h3</h3>
<h4>제목 h4</h4>
<h5>제목 h5</h5>
<h6>제목 h6</h6>

<head>

  • 해당 문서에 대한 정보인 메타데이터의 집합을 정의할 때 사용한다.
  • HTML의 head는 페이지를 열 때 웹 브라우저에 표시되지 않는다. head의 내용이 하는 일은 페이지에 대한 metadate를 포함하는 것이다.
  • <title>에서 사용하는 태그들은 다음과 같다.
    <style>
    <base>
    <link>
    <meta>
    <script>
    <noscript>
    <favicon>
  • script와 noscript는 body 안에 들어갈 수 있다.
<head>
    <title>문서의 제목<title>
</head>

<hr>

  • HTML 문서에서 내용을 구분하거나 주제의 변화를 정의한다.
  • 주제가 바뀔 때 수평 가로선으로 정의할 때 사용한다.
<div>
    <p>hr은 수평 가로선으로 정의합니다.</p>
    <hr>
    <p>hr은 수평 가로선으로 정의합니다.</p>
</div>

<html>

  • HTML 문서의 루트 요소(root element)를 정의할 때 사용
  • DOCTYPE 선언을 제외한 모든 다른 HTML 요소를 포함하기 위한 컨테이너이며, 브라우저에게 해당 문서가 HTML 문서임을 알려주는 역할을 한다.
<!DOCTYPE html>
<html lang="ko">
    <head>
        <title>HTML 문서의 제목입니다.</title>
    </head>
    <body>
        <p>HTML 문서의 콘텐츠(contents) 영역입니다.</p>
    </body>
</html>
