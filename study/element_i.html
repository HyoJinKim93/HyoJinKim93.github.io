Element_i

<i>

  • 기본 텍스트와는 다른 분위기나 음성을 위한 텍스트 영역을 정의할 때 사용
  • 이탤릭체로 표현되며, 전문용어나 관용구, 문어체, 서양 문헌에 나오는 배 이름 등을 나타낼 때 사용된다.
  • 다음과 같은 의미 요소(semantic element) 중에서 사용하기에 적당한 요소를 찾지 못했을 때만 i 요소를 사용할 수 있다.
    • cite : 창작물의 제목
    • dfn : 용어의 제목
    • em : 강조된 텍스트
    • mark : 하이라이트된 텍스트
    • strong : 중요한 텍스트
  • HTML4에서 i는 이텔릭체로 표현되었지만, HTML5에서는 반드시 이텔릭체로 표현되지 않는다.
<p>
    <i>타이타닉호</i>는 영국의 화이트 스타 라인이 운영한 북대서양 횡단 여객선이다.
</p>

타이타닉호는 영국의 화이트 스타 라인이 운영한 북대서양 횡단 여객선이다.

카르페 디엠(Carpe diem)은 오늘을 즐기라는 뜻이다.

<iframe>

  • 인라인 프레임을 정의할 때 사용한다.
  • 인라인 프레임은 현재 HTML 문서에 다른 문서를 포함시킬 때 사용한다.
  • iframe 요소의 시작 태그와 종료 태그 사이에는 iframe을 지원하지 않는 브라우저를 위한 텍스트를 포함할 수 있다.
  • HTML5에선 iframe 태그의 align, frameborder, longdesc, marginheight, marginwidth, scrolling 속성이 더 이상 지원되지 않도록 변경,
    sandbox 속성과 srcdoc 속성이 새롭게 추가 되었다.
  • XHTML에서는 name 속성을 더 이상 지원하지 않으므로, id 속성을 대신 사용한다.
<iframe src="http://tcpschool.com"></iframe>
필수 속성
속성명 속성값 설명
align left, right, top, middle, bottom 주변 요소에 따라 iframe 요소가 정렬되는 방법을 명시한다
HTML5에서는 더 이상 지원하지 않음
frameborder 0, 1 테두리(border)를 보여줄지 여부를 명시함
HTML5에서는 더 이상 지원하지 않음
height 픽셀 높이를 명시함
longdesc URL 보일 콘텐츠에 대한 설명을 담고 있는 페이지를 명시함
HTML5에서는 더 이상 지원하지 않음
marginheight 픽셀 콘텐츠의 위아래 마진(margin)을 명시함
HTML5에서는 더 이상 지원하지 않음
marginwidth 픽셀 콘텐츠의 좌우 마진(margin)을 명시함
HTML5에서는 더 이상 지원하지 않음
name 텍스트 iframe 요소의 이름을 명시함
sandbox 비어 놓음 : 모든 제한 사항(restrictions)을 적용함 allow-forms : 리소스가 폼 데이터를 제출할 수 있도록 허용
allow-modals : 리소스가 모달 윈도우를 열 수 있도록 허용
allow-orientation-lock : 리소스가 화면 방향 전환을 잠글 수 있도록 허용
allow-pointer-lock : 리소스가 Pointer Lock API를 사용할 수 있도록 허용
allow-popups : window.open()이나 target=“_blank”, showModalDialog() 등의 팝업(popup)을 허용
allow-popups-to-escape-sandbox : 모든 제한 사항이 적용된 문서에서 새로운 창을 열 때 제한 사항을 상속받지 않은 창을 열 수 있도록 허용
allow-presentation : 리소스가 프리젠테이션 세션을 시작할 수 있도록 허용
allow-same-origin :
allow-scripts : 리소스가 스크립트를 실행할 순 있지만, 팝업은 생성하지 못 함
allow-storage-access-by-user-activation : 리소스가 Storage Access API를 사용하여 상위 스토리지 기능에 접근 요청을 할 수 있도록 허용
allow-top-navigation : 리소스가 최상위 브라우징 컨텍스트 (_top)을 탐색할 수 있도록 허용
allow-top-navigation-by-user-activation : 리소스가 사용자의 요청이 있을 때만 최상위 브라우징 컨텍스트(_top)를 탐색할 수 있도록 허용 		콘텐츠에 대한 추가적인 제한 사항(restrictions)들의 집합을 명시함
scrolling yes, no, auto scrollbar를 표시할지 여부를 명시
HTML5에서는 더 이상 지원하지 않는다.
src URL iframe 요소에 보인 문서의 URL을 명시
srcdoc HTML 코드 iframe 요소에 보일 웹 페이지의 HTML 코드를 명시
width 픽셀 iframe 요소의 너비
